# valkey

Valkey ist ein hochleistungsfähiger Datenstrukturserver, der in erster Linie Key/Value-Workloads bedient. Im MLA wird er im Kontext Quantumleap eingesetzt.

## Voraussetzungen

* Kubernetes 1.23+
* Helm 3.8.0+
